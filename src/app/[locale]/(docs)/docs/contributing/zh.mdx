## 如何贡献 Latent Box

感谢你有兴趣贡献该项目。在你提交第一个 pull request 之前，请你花一些时间阅读这个文档。我们建议先检查现有的 issue 和 pull request 看看有没有人正在做和你想做的类似的事情。

如果你需要帮助，请联系 [@cpunisher](https://twitter.com/cpunisher)。

### Fork 这个项目

你可以在 GitHub 项目主页 Fork 这个项目到你的账号下。

### Clone 到本地

```bash
git clone https://github.com/your-username/latentbox.git
```

### 切换到项目目录

```bash
cd latentbox
```

### 创建新的分支

```bash
git checkout -b my-new-branch
```

### 安装依赖

```bash
yarn
```

### 修改代码并创建 pull request

接下来添加或修改一些代码，并在 Github 上创建 pull request。

当 pull request 被接受时，我们会将其合并到主分支，并将你添加为贡献者。
